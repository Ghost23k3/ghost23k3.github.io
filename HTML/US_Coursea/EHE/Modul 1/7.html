Payment Card Industry Data Security Standard (PCI DSS)

If you work for an organization that uses and processes Customers Credit Card Data, then you are covered by: Payment Card Industry Data Security Standard (PCI DSS) = Credit Card/Debit Card Security standard.

  • They want, that we build and maintain a secure Network
  • Protect Cardholder Data
  • Maintain a Vulnerability Management Program
  • Implements Strong Access Control Measures
  • Regulary Monitor and Test Networks
  • Maintain an Information Security Policy
